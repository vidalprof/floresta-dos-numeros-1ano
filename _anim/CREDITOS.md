# Créditos das imagens (arte / sprites)

Este projeto usa arte livre. Abaixo, os autores e as licenças — **dar crédito é
obrigatório** para os assets LPC (CC-BY-SA / GPL). Assets CC0 não exigem crédito,
mas os listamos por transparência.

## LPC — Liberated Pixel Cup (licença: CC-BY-SA 3.0 e/ou GPL 3.0)
Uso livre (inclusive escolar), **com atribuição** e mantendo derivados sob a mesma
licença livre.

- **LPC Base Assets** (chão/grama, árvores, água, princesa, soldado, props):
  autor principal **Lanea Zimmerman (Sharm)** e colaboradores do LPC.
  Fonte: OpenGameArt — "Liberated Pixel Cup (LPC) Base Assets".
- **Universal LPC Spritesheet** (camadas de personagem: corpo, cabelo, roupas):
  colaboradores do LPC (projeto Universal-LPC-Spritesheet). Fonte: GitHub
  (jrconway3 / LiberatedPixelCup). Licença CC-BY-SA 3.0 / GPL 3.0.

## CC0 — domínio público (sem obrigação de crédito)
- **Ninja Adventure Asset Pack** — **Pixel-Boy** e **AAA** (CC0). Fonte:
  itch.io / GitHub (pixel-boy/NinjaAdventure). *(usado se/quando entrar no projeto.)*
