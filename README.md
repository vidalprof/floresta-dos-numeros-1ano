# A Floresta dos Números — 1º ano

Site educativo publicado no GitHub Pages:
**https://vidalprof.github.io/floresta-dos-numeros-1ano/**

Este repositório também guarda a **"Fábrica de Sites"**, que cria e atualiza
páginas no GitHub Pages de forma automática.

---

## 📌 Lembrete: como usar a Fábrica em qualquer sessão

Cada sessão do assistente começa "do zero" e não lembra das conversas
anteriores. As instruções completas ficam no arquivo **`CLAUDE.md`** deste
repositório, que o assistente lê sozinho **quando a sessão é aberta aqui**.

**Se você abrir uma sessão em outro repositório**, é só colar esta frase para
o assistente se situar:

> As instruções da Fábrica de Sites estão no `CLAUDE.md` do repositório
> `vidalprof/floresta-dos-numeros-1ano`. Leia de lá antes de agir.

### Para criar um site novo, diga ao assistente:
1. **O nome** do site (só minúsculas, números e hífens — ex.: `tabuada-divertida`)
2. **Envie o arquivo HTML**

O assistente cria o repositório, publica e devolve o link no padrão
`https://vidalprof.github.io/NOME-DO-SITE/`.

### Para atualizar este site:
Envie o novo `index.html` e peça para atualizar — ele se republica sozinho
em cerca de 1 minuto.
