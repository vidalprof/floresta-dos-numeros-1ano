## O problema (honesto)
As atividades premium existentes (ex.: "O Grande Circo do Teo") e o **EducaVerso** usam o modelo
antigo: quiz, completar, arrasta-e-solta, ligar colunas. A ciência da aprendizagem (ver
`MODELO-APRENDIZAGEM-EDUVERSE.md`) mostra que esse formato é o **fraco** — a "prova disfarçada" /
"brócolis com chocolate": engaja, mas o ganho de aprendizado é modesto. Elas **contradizem** a
filosofia nova (problema primeiro, aprender fazendo, o mundo precisa da criança, o Byte pergunta).

## Nuance importante (NÃO jogar tudo fora)
- Nem toda interação antiga é lixo. Uma pergunta de **recuperação** (lembrar/produzir) bem feita É
  **retrieval practice** — que a ciência ADORA. O problema é quando o **quiz É a atividade** (prova
  disfarçada), não quando uma pequena checagem de recuperação **apoia** um núcleo de resolver/construir.
- Então "atualizar" = **mudar o NÚCLEO** de quiz → construir/resolver, e **rebaixar** o quiz a um
  elemento de apoio (recuperação espaçada), não apagar tudo.

## A boa notícia: construir o novo modelo É o caminho de atualizar as antigas
O **gerador (Fábrica) + pedagogo + motor adaptativo** que estamos construindo é exatamente a
ferramenta que permite **REGERAR** as atividades antigas no modelo novo, com eficiência (em vez de
refazer uma a uma na mão). Ou seja: prioridade é terminar o novo motor; a atualização das antigas
vem "de brinde" depois.

## Plano de atualização (quando chegar a hora)
1. **Inventariar** as atividades antigas (premium + EducaVerso) e classificar: (a) núcleo já é
   resolver/construir → mantém; (b) núcleo é quiz/arrasta-solta → refazer o núcleo; (c) baixo uso →
   pode aposentar.
2. **Priorizar por valor/uso** — atualizar primeiro as mais usadas/importantes; o resto espera.
3. **Regerar** as de núcleo-quiz pelo novo pipeline (pedagogo → dinâmica concreta → roteirista →
   motor adaptativo + medição), reaproveitando tema/personagens/arte que já existem.
4. **Passar pelos portões** (0, 0.5-Vygotsky, integração intrínseca, QA, encantamento, duração) —
   como qualquer atividade nova.
5. **Medir** que a versão nova ensina mais (transferência+retenção), não só que é bonita.
